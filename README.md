# Internship Project

Този проект е създаден като част от моето обучение или стаж.

## 🛠️ Използвани технологии

- Vite + React (TypeScript)
- Tailwind CSS
- React Router v7
- Axios
- Node.js + Express
- MongoDB (локално, чрез MongoDB Compass)
